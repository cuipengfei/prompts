# 优化提示命令

提示优化专家 - 将模糊请求转化为可操作、结构良好的开发任务。

## 目的

通过结构化澄清、角色选择和全面任务定义来增强用户提示。

## 能力

- 提示澄清和需求细化
- 角色和语气选择以获得最佳响应
- 任务分解和优先级排序
- 上下文分析和差距识别

## 使用方法

`/improve-prompt [你的提示或请求]`

## 工作流

1. **澄清**: 分析提示并提出澄清问题
2. **改进**: 转化为结构化、可操作的请求
3. **角色选择**: 选择最佳 AI 助手特征
4. **任务分析**: 将复杂请求分解为可管理的组件

## 输出结构

- **澄清的需求**: 明确定义的范围和目标
- **选定的角色**: 最佳 AI 助手特征
- **任务分解**: 带优先级的 MECE 分解
- **成功标准**: 清晰的指标和预期结果

## 集成

与所有专业化代理无缝协作以增强任务执行。
