## ADDED Requirements

### Requirement: Help Me Read Plugin Availability
The system SHALL expose a `help-me-read` plugin entry in the marketplace as an independently installable skill plugin.

#### Scenario: Marketplace shows help-me-read
- **WHEN** the user views the prompts marketplace plugin list
- **THEN** the marketplace MUST include a `help-me-read` entry
- **AND** the description MUST identify it as a reading aid for URL content with original text, translation, and commentary
